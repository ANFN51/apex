# Apex Realty Centers Website

This repository contains the source code for the Apex Realty Centers website, a premium real estate brokerage in Detroit and Garden City, MI.

## Features
- Responsive design with Bootstrap
- Parallax sections and AOS animations
- Mortgage calculator
- Dynamic testimonials
- Contact form with Formspree integration
- Dark mode toggle

## Setup
1. Clone the repo
2. Open index.html in a browser
3. For production, deploy to GitHub Pages or similar
